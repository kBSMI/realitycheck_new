# SMI IP Boundary Note

**Version:** 1.0  
**Classification:** Confidential — Pilot Review Only

---

## Intellectual Property Status

Symbolic Memory Infrastructure (SMI) is a pre-existing framework and proprietary intellectual property. The concepts, scoring architecture, event taxonomy, service design, and continuity assurance methodology embodied in this pilot are prior work and are not contributed to the public domain by virtue of this demonstration.

---

## Scope of This Pilot Material

This pilot console and its accompanying documentation use synthetic data and mock event structures throughout. No live AI infrastructure, production data, proprietary customer data, or confidential third-party systems are used or referenced.

The pilot is designed to demonstrate the SMI framework's capabilities and integration architecture for evaluation purposes only.

---

## F5 ADSP Simulation Connector

The F5 ADSP Simulation Connector is a demonstration layer using mock event structures to illustrate how SMI could interoperate with AI gateway, guardrail, red-team, and XOps event streams. It is not an official F5 integration. F5 Networks, BIG-IP, and ADSP are trademarks of F5, Inc. No partnership, endorsement, or official relationship between SMI and F5 is implied or represented by this simulation layer.

## Ownership and Transfer

Viewing, evaluating, or using this pilot console does not transfer ownership of the underlying SMI framework, scoring methodology, service architecture, or any associated intellectual property.

Pilot outputs — including scores, drift findings, audit records, and reports — are generated from synthetic data for demonstration purposes. They do not constitute deliverables that transfer SMI framework ownership unless separately agreed in writing under a formal pilot agreement or licensing arrangement.

Recipients of this pilot material are expected to treat it as confidential pre-NDA review material. This includes the scoring logic, service architecture documentation, event schema, and test plan.

Redistribution, publication, or use of this material outside the context of evaluating the SMI pilot requires written consent.
